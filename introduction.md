# STAMINA REST API Documentation

The following

+--------+-----------------+-----------------------------+
| Method | URL             | Description                 | 
+--------+-----------------+-----------------------------+
| GET    | /jobs/[job id]  | Gets the status for a job   |
| GET    | /jobs/          | Lists jobs (priviledged)    |
| POST   | /jobs           | Creates a job, returns ID   |
| DELETE | /jobs/[job id]  | Stops a Job                 |
+--------+-----------------+-----------------------------+
